# FE5213 Project Group 4

This repository contains the working notebooks and exported figures for our FE5213 macro project.

The project is organized so that the main notebooks can be run from top to bottom without needing to jump across files.

## Which File Answers Which Section

- `ricardian_equivalence_3.1.ipynb`
  Covers Section 3.1 of the project.
- `fiscal_policy_experiments_3.2_3.3.ipynb`
  Covers Sections 3.2 and 3.3 of the project.

If you are reading or reproducing the report section by section, this is the intended mapping.

## Recommended Run Order

1. Run `ricardian_equivalence_3.1.ipynb` from the first cell to the last cell for Section 3.1.
2. Run `fiscal_policy_experiments_3.2_3.3.ipynb` from the first cell to the last cell for Sections 3.2 and 3.3.

Each notebook is designed to be self-contained in the sense that it should execute sequentially from top to bottom.

## Repository Structure

## Dependencies

Use Python 3 with the following packages:

- `numpy`
- `scipy`
- `matplotlib`
- `pandas`

Optional:

- `quantecon`

The fiscal notebook includes a fallback for the helper section where `quantecon` may otherwise be used.

## How to Run

1. Open the notebook for the section you want to reproduce.
2. Run all cells in order, from top to bottom.
3. Check the `plots/` folder for exported figures after execution.

For full reproduction of the written project:

1. Run `ricardian_equivalence_3.1.ipynb`.
2. Run `fiscal_policy_experiments_3.2_3.3.ipynb`.
